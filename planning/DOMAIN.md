# DOMAIN — the client's world

The goal of this file is to make the Builder speak the **client's language**, not generic software language.

## Users / roles

- Computer Nerds
- Geeks
- Old People
- Nation State Actors
- People with too much time on their hands.

## Current process (what happens today, manually)

There is no manual process. This is a greenfield implementation.

## Terminology

- BBS - Bulletin Board System
- Message Board - Place for people to leave messages with each other.
- Door Games - Dumb Text Games that can be launched from inside the BBS
- File Library - Library of objects users can download to their computers
- Membership

## Business rules / invariants

- A BBS is a system used to connect people across the internet with common interests.
